# Rick and Morty characters

Showcase Project - React, TypeScript with styled-components (extra Plop with some Handlebars).

## Demo

Project is available at [Link](https://vercel.com/klosiek/worldcloudgame)

## Start

```
yarn
yarn start
```

or

```
npm i
npm run start
```
